# E-commerce Product Catalogue
A full-stack web application for managing a product storefront.

## Features
- Dynamic product listing from MySQL database
- Category-based filtering
- Admin panel for managing orders and products
- Responsive UI with a Gold and White aesthetic

## Tech Stack
- Frontend: HTML5, CSS3, JavaScript
- Backend: PHP
- Database: MySQL# ecommerce-shop
